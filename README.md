# Python ML December Sprint

A structured 30-day sprint focused on mastering Python, Data Structures, Exploratory Data Analysis, Machine Learning, Deep Learning, Natural Language Processing, and MLOps.  
This repository documents daily hands-on progress through exercises, code implementations, notes, mini-projects, and end-to-end machine learning workflows.

---


## Each day includes:
- Exercises  
- Code solutions  
- Notes  
- Mini-projects  

---

## 30-Day Breakdown

### Days 1–7: Python + DSA + Pandas/NumPy
- Python fundamentals
- Data structures
- OOP
- Hashmaps, two pointers, sliding window
- Pandas and NumPy
- Exploratory Data Analysis

### Days 8–15: Machine Learning
- Regression and classification algorithms  
- Decision Trees, Random Forests, SVM, KNN  
- Clustering, PCA  
- XGBoost, LightGBM  
- Model evaluation, tuning, pipelines  

### Days 16–22: Deep Learning (PyTorch & TensorFlow)
- Neural networks  
- CNNs, RNNs, LSTMs  
- Transfer learning  
- Computer vision and time-series models  

### Days 23–27: NLP (BERT, GPT, Transformers)
- Text preprocessing  
- Embeddings  
- Fine-tuning BERT  
- Summarisation, classification, semantic search  

### Days 28–30: MLOps
- FastAPI model serving  
- Docker  
- MLflow  
- AWS deployment (EC2, Lambda, S3, SageMaker)  

---

## Goals

By the end of the sprint:
- Become proficient in Python for data science and machine learning  
- Build competency in coding interviews (LeetCode)  
- Understand and implement core ML algorithms  
- Apply deep learning using PyTorch and TensorFlow  
- Build and deploy end-to-end machine learning systems  
- Maintain consistent daily coding discipline and version control practices
